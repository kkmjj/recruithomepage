나만의 취업정보 홈페이지 만들어보기 

개발 이유  : 취업준비를 하며 웹 개발자로서 실력도 쌓고 정말 필요한 것을 직접 개발해보고 싶어 프로젝트 진행
팀원 : 대학교 동기 외 1명 
내용(기능) - 자소설 닷컴이라는 사이트에 있는 날짜별 기업정보를 웹크롤링 - python beautifulsoup을 이용하여 웹크롤링
	    -mysql에 pysql을 이용하여 크롤링 한 데이터를 저장 
		-> python을 이용한 이유 : jsp에서 보다 python을 통한 웹크롤링이 좀 더 많이 쓰이고, python도 같이 공부하기 위해 적용 

              - 저장한 정보를 jsp형태로 만들어둔 홈페이지 html 안에 내용 삽입
              -  로그인 로그아웃 / 회원가입 기능 -> 특정 로그인을 통해서 웹 서비스를 이용
